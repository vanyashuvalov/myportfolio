## Project Overview

A bold redesign of my portfolio website using a desktop operating system metaphor. Features draggable widgets, project folders, and interactive elements that make browsing feel like exploring a digital workspace.

::: callout info
This project showcases my ability to create unique, memorable web experiences without relying on frameworks.
:::

## Design Philosophy

Instead of a traditional scrolling portfolio, I wanted to create something interactive and playful. The desktop metaphor allows visitors to:
- Explore at their own pace
- Interact with widgets and elements
- Discover hidden features
- Experience personality through design

## Key Features

### Desktop Canvas
- Draggable widgets with physics
- Persistent positioning
- Smooth animations
- Responsive grid system

### Widget System
- Modular architecture
- Event-driven communication
- Reusable base class
- Easy to extend

### Project Showcase
- Folder metaphor for categories
- Modal-based project details
- Markdown content support
- Custom block components

::: stats
- **0** frameworks used
- **100%** vanilla JavaScript
- **Figma** for design
- **Feature-Sliced Design** architecture
:::

## Technical Stack

Built entirely with vanilla web technologies:
- **HTML5** - Semantic markup
- **CSS3** - Custom properties, Grid, Flexbox
- **JavaScript ES6+** - Modules, classes, async/await
- **Python FastAPI** - Backend API
- **Markdown** - Content management

## Design System
